# Spring Boot CRUD Example with MySQL

This project demonstrates a simple **REST API** for Employee Management using Spring Boot, MySQL, JPA, and Hibernate.

## Features
- Create Employee (POST)
- Get Employees (GET)
- Update Employee (PUT)
- Delete Employee (DELETE)

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## API Mthods
- GET
- POST
- PUT
- DELETE
